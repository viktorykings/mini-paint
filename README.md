```

```


### Database Snapshot
Firebase `paints` collection has fields:
- author: contains author's name
- url: contains image's url
- authorID: contains author's id


### Application stack
- Vite
- React
- Typescript
- Redux, redux toolkit
- React Router
- Firebase
- React-colorful
